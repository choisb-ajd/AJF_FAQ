적립 불가 기준

이 경우엔 광고비가
적립되지 않아요

아래 5가지 예외 기준을 미리 확인하고 신청해주세요.

1
다른 비교견적 채널 이용 후 아정당에서 가입한 경우
타 사이트나 앱에서 이미 견적을 조회한 후 아정당으로 가입하면 광고비가 적립되지 않아요.
💡 고객에게 "처음부터 아정당 링크로만 가입하세요"라고 안내해주세요.
2
계약 후 1개월 내에 취소된 경우
계약 체결 후 1개월 이내 취소 시, 이미 적립·출금된 광고비는 이후 적립금에서 차감될 수 있어요.
⚠️ 출금까지 완료된 광고비도 환수 대상이 될 수 있으니 주의하세요.
3
책임보험 또는 단기보험 가입인 경우
책임보험(의무보험)과 단기보험은 광고비 지원 대상에서 제외돼요.
4
이륜차(오토바이) 가입인 경우
이륜자동차 보험은 광고비 지원 대상에서 제외돼요.
5
갱신 시 기존과 동일한 보험사로 재가입한 경우
보험 갱신 시 기존 보험사와 동일한 보험사로 다시 가입하면 광고비 적립이 되지 않아요.
💡 다른 보험사로 변경하면 광고비 적립이 가능해요! 매니저가 최저가 보험사로 안내해드려요.
간혹 발생하는 미적립 케이스

기준에 맞는데도
적립이 안 됐나요?

아래 실수들 때문인 경우가 많아요. 확인해보세요.

✏️
고객 정보 오기재
고객 성명란에 딜러 본인 이름을 적거나, 고객 정보를 잘못 입력한 경우
예) 고객 성명란에 "홍길동(딜러)" 입력 → 미적립
🔢
차량번호 입력 오류
차량번호 띄어쓰기, 고객 성함이나 전화번호 오기재
예) "12가 3456" (띄어쓰기 포함) → 미적립 가능성
🚗
동일 명의 여러 대 차량 중 일부만 접수
같은 명의의 여러 대 차량 중 1대만 접수한 경우, 나머지 차량은 별도 접수가 필요해요.
추가 접수가 필요할 경우 고객센터에 별도 연락!
🔍
차대번호 vs 차량번호 불일치
차대번호로 가입 요청했지만 보험사에서 차량번호로 조회되는 경우
차량번호 기준으로 재등록이 필요할 수 있어요.
🏢
법인 가입 시 연락처 미기재
법인 명의로 접수하면서 고객 연락처를 기재하지 않은 경우
법인 가입은 반드시 전화 가입으로, 고객 연락처 필수!
🌐
직접 보험사 접속 후 가입
고객이 아정당 링크를 거치지 않고 직접 보험사 홈페이지에서 가입한 경우
반드시 아정당 전용 링크로 접속 후 가입해야 해요!
신청 전 체크리스트

광고비 100% 받기 위한
신청 전 확인 사항

이것만 지키면 미적립 걱정 없어요!

✅

아래 항목을 모두 확인했다면 광고비 적립 OK!

신청 전 딜러님이 직접 체크해주시면 누락 없이 광고비를 받으실 수 있어요.

📋 신청 전 필수 체크리스트

✓
고객 성명이 정확한지 확인 (띄어쓰기 없이)
✓
고객 연락처가 정확한지 확인
✓
차량번호가 정확한지 확인 (띄어쓰기 없이)
✓
고객이 다른 채널에서 견적을 보지 않았는지 확인
✓
책임보험, 단기보험, 이륜차가 아닌지 확인
✓
갱신 시 다른 보험사로 변경하는지 확인
✓
법인 가입인 경우 고객 연락처 기재 여부 확인
✓
고객이 아정당 전용 링크로만 접속하도록 안내

기준에 맞는데 미적립됐나요?

전담 매니저 또는 고객센터로 바로 연락해 주시면 확인해드려요.
이름·연락처·고객 정보를 함께 알려주시면 더 빠르게 처리해드려요.
